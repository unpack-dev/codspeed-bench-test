# bench-demo

[![CodSpeed](https://img.shields.io/endpoint?url=https://codspeed.io/badge.json)](https://codspeed.io/hardfist/bench-demo?utm_source=badge)

A small Rust benchmark playground with:

- string repeat micro-benchmarks
- single-file JavaScript parser benchmarks
- a `rayon`-powered benchmark that parses the unpacked `three@0.183.2` ESM source tree checked into `benches/fixtures/threejs/`
